pre {target}: put the cursor before the target word
post {target}: put the cursor after the target word
take {target} : select the target word
take color [target]: select the color target word
take harp past drum: select the word defined by harp through to the word defined by drum
take {target1} and {target2}: select both target one and target two
chuck trap: delete the word marked by trap
chuck red trap: chuck the word marked by red trap
chuck line number : chuck that line number
take line: select the current line
take row 3: select row three
take row {n} past row {m}
row 3 : move the cursor to row three
copy {target}: copy the target word or selection
paste: paste the clipboard contents at the cursor
swap {target1} and {target2}: swap the positions of two targets
rename {target} to {new}: rename the target to a new value
replace {target} with {text}: replace the target with specified text
select all: select the entire document
undo: undo the last action
redo: redo the last undone action
comment {target}: comment out the target
uncomment {target}: remove comment from the target
